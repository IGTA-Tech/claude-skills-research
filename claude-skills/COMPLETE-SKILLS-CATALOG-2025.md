## 🚀 Installation

### Option 1: Individual Skills

Download individual skills from the OneWave AI or Anthropic directories and install them into your Claude Code skills folder:

```bash
# macOS/Linux
~/.claude/skills/

# Windows
%APPDATA%\Claude\skills\
```

### Option 2: Bulk Installation (OneWave AI Skills)

```bash
# Clone or download the repository
cd claude-skills

# Copy all OneWave AI skills
cp *.zip ~/.claude/skills/

# Extract all skills
cd ~/.claude/skills/
unzip "*.zip"
```

### Option 3: Official Anthropic Skills

Official Anthropic skills can be installed from their GitHub repository:

```bash
git clone https://github.com/anthropics/claude-code-skills.git
cp -r claude-code-skills/* ~/.claude/skills/
```
