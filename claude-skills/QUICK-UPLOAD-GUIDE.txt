✅ OPTION 1: CLAUDE CODE (CLI) - RECOMMENDED
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Skills activate AUTOMATICALLY based on your requests!

STEP 1: INSTALL CLAUDE CODE
────────────────────────────────────────────────────────────────────────────
  Visit: https://claude.ai/download

  Or install via npm:
  $ npm install -g @anthropics/claude-code

STEP 2: INSTALL A SKILL (30 SECONDS)
────────────────────────────────────────────────────────────────────────────
  # Extract skill
  $ unzip screenshot-to-code.zip

  # Create skills directory
  $ mkdir -p ~/.claude/skills

  # Copy skill
  $ cp -r screenshot-to-code ~/.claude/skills/

  # Verify
  $ ls ~/.claude/skills/

STEP 3: USE IT!
────────────────────────────────────────────────────────────────────────────
  # Start Claude Code
  $ claude

  # Just ask naturally - skill activates automatically!
  > "Convert this screenshot to React code"
  > "Optimize this blog post for SEO"
  > "What does Reddit think about this?"

  No commands needed - Claude knows when to use each skill!

🌐 OPTION 2: CLAUDE.AI WEB (PROJECTS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️  Skills don't auto-activate on web - you reference them manually

STEP 1: CREATE A PROJECT
────────────────────────────────────────────────────────────────────────────
  1. Go to https://claude.ai
  2. Click "Projects" in sidebar
  3. Click "Create Project"
  4. Name it (e.g., "Claude Skills Library")

STEP 2: UPLOAD SKILL FILES
────────────────────────────────────────────────────────────────────────────
  1. Click "Add content" → "Upload files"
  2. Upload SKILL.md files from extracted folders
  3. Or copy/paste content as text

  Example:
    - Upload: screenshot-to-code/SKILL.md
    - Upload: seo-optimizer/SKILL.md
    - Upload: README.md (for reference)

STEP 3: USE IN CONVERSATION
────────────────────────────────────────────────────────────────────────────
  You need to MANUALLY reference the skill:

  ❌ Won't work: "Convert this screenshot to code"

  ✅ Works: "Using the screenshot-to-code instructions from
             project knowledge, convert this image to React code"

SKILL NOT WORKING?

1. CHECK FILE LOCATION
   $ ls ~/.claude/skills/screenshot-to-code/SKILL.md

   Should show the file exists.

2. RUN WITH DEBUG MODE
   $ claude --debug

   This shows skill loading errors.

3. BE MORE EXPLICIT
   Instead of: "Look at this"
   Say: "Convert this screenshot to React code using screenshot-to-code"

4. CHECK YAML SYNTAX
   $ head -10 ~/.claude/skills/screenshot-to-code/SKILL.md

   Should show:
   ---
   name: screenshot-to-code
   description: Convert UI...
   ---
