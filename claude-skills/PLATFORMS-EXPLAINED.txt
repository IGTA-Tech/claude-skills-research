✅ CLAUDE CODE (WHERE SKILLS WORK)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WHAT IS CLAUDE CODE?

Claude Code is Anthropic's official CLI tool that:
  • Runs in your terminal
  • Has access to your filesystem
  • Can read/write/edit files
  • Executes bash commands
  • Integrates with your development workflow

INSTALLATION:

  Method 1: Direct Download
  ─────────────────────────────────────────────────────────────────────────
  Visit: https://claude.ai/download
  Download for macOS/Windows/Linux

  Method 2: NPM
  ─────────────────────────────────────────────────────────────────────────
  $ npm install -g @anthropics/claude-code

  Verify:
  ─────────────────────────────────────────────────────────────────────────
  $ claude --version

SKILLS IN CLAUDE CODE:

  Skills are stored in:
  ─────────────────────────────────────────────────────────────────────────
  ~/.claude/skills/          ← Personal (all projects)
  .claude/skills/            ← Project-specific

  Installation:
  ─────────────────────────────────────────────────────────────────────────
  $ mkdir -p ~/.claude/skills
  $ unzip screenshot-to-code.zip
  $ cp -r screenshot-to-code ~/.claude/skills/

  Usage:
  ─────────────────────────────────────────────────────────────────────────
  $ claude
  > "Convert this screenshot to React code"

  ✅ Skill activates AUTOMATICALLY based on your request!

HOW SKILLS WORK IN CLAUDE CODE:

  ┌─────────────┐      ┌──────────────┐      ┌─────────────┐
  │   You Ask   │ ───> │ Claude Reads │ ───> │ Skill       │
  │ "Convert    │      │ Skill Files  │      │ Activates   │
  │ screenshot" │      │ in ~/.claude │      │ & Executes  │
  └─────────────┘      └──────────────┘      └─────────────┘

  1. You make a request
  2. Claude scans installed skills
  3. Finds matching skill based on description
  4. Follows skill instructions
  5. Completes task automatically

WHAT IS CLAUDE.AI?

The web interface at https://claude.ai with:
  • Chat conversations
  • Projects (knowledge bases)
  • Artifacts (code/document generation)
  • NO automatic skills

PROJECTS (NOT SKILLS):

Projects let you upload documents for Claude to reference:

  Create Project:
  ─────────────────────────────────────────────────────────────────────────
  1. Go to https://claude.ai
  2. Click "Projects" sidebar
  3. "Create Project"
  4. Name: "Claude Skills"

  Upload Knowledge:
  ─────────────────────────────────────────────────────────────────────────
  1. Click "Add content"
  2. Upload SKILL.md files
  3. Add documentation

  Use in Chat:
  ─────────────────────────────────────────────────────────────────────────
  "Using the screenshot-to-code instructions from project knowledge,
   convert this image to React code."

  ⚠️  You must MANUALLY reference the instructions!

HOW PROJECTS WORK:

  ┌─────────────┐      ┌──────────────┐      ┌─────────────┐
  │   You Ask   │ ───> │   You Must   │ ───> │ Claude Uses │
  │ & Reference │      │   Explicitly │      │ Project     │
  │ Project KB  │      │   Mention It │      │ Knowledge   │
  └─────────────┘      └──────────────┘      └─────────────┘

🎯 WHICH SHOULD YOU USE?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

USE CLAUDE CODE IF:
  ✅ You're a developer
  ✅ You work in terminal/CLI
  ✅ You want automatic skill activation
  ✅ You need file editing capabilities
  ✅ You want to manage multiple skills
  ✅ You prefer keyboard-based workflow
  ✅ You want git-based sharing
  ✅ You need terminal commands (git, npm, etc.)

USE CLAUDE.AI WEB IF:
  ✅ You prefer browser interface
  ✅ You don't need CLI tools
  ✅ You're okay with manual references
  ✅ You want quick access (no install)
  ✅ You need to share via links
  ✅ You prefer mouse-based interaction

USE BOTH IF:
  ✅ Claude Code for development work
  ✅ Claude.ai for quick questions
  ✅ Projects for team knowledge bases
  ✅ Skills for personal automation

FOR DEVELOPERS:

  1. Install Claude Code
     ─────────────────────────────────────────────────────────────────────
     $ npm install -g @anthropics/claude-code

  2. Install All Skills
     ─────────────────────────────────────────────────────────────────────
     $ cd /path/to/claude-skills
     $ mkdir -p ~/.claude/skills
     $ for file in *.zip; do unzip -q "$file"; done
     $ cp -r */  ~/.claude/skills/

  3. Use Daily
     ─────────────────────────────────────────────────────────────────────
     $ cd your-project
     $ claude
     > "Convert this screenshot to React"
     > "Review this code for security"
     > "Optimize this content for SEO"

FOR NON-DEVELOPERS:

  1. Use Claude.ai Web
     ─────────────────────────────────────────────────────────────────────
     Visit: https://claude.ai

  2. Create Project
     ─────────────────────────────────────────────────────────────────────
     Projects → Create → Name: "Skills Library"

  3. Upload Knowledge
     ─────────────────────────────────────────────────────────────────────
     Add content → Upload SKILL.md files

  4. Reference in Chat
     ─────────────────────────────────────────────────────────────────────
     "Using the [skill name] instructions from project knowledge,
      help me with [task]"
