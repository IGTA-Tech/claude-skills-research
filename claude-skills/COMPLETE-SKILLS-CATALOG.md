## ⚡ Installation Guide

### Install OneWave AI Custom Skills

```bash
# Personal skills (all projects)
mkdir -p ~/.claude/skills
unzip screenshot-to-code.zip
cp -r screenshot-to-code ~/.claude/skills/
```

### Install Official Anthropic Skills

```bash
# Copy from official-anthropic-skills directory
mkdir -p ~/.claude/skills
cp -r official-anthropic-skills/algorithmic-art ~/.claude/skills/
cp -r official-anthropic-skills/artifacts-builder ~/.claude/skills/
# ... copy whichever skills you want
```

### Install All Skills at Once

```bash
# OneWave AI custom skills
cd /path/to/claude-skills
for file in *.zip; do
  unzip -q "$file"
  cp -r "${file%.zip}" ~/.claude/skills/
done

# Official Anthropic skills
cp -r official-anthropic-skills/* ~/.claude/skills/

# Verify
ls ~/.claude/skills/
```
