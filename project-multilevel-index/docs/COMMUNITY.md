## 🎯 社区准则

为了维护一个友好、专业的社区环境,请遵守以下准则:

1. **互相尊重** - 尊重他人的观点和经验
2. **建设性反馈** - 提供有价值的建议和意见
3. **分享知识** - 帮助新用户解决问题
4. **保持专业** - 避免与项目无关的讨论
5. **遵守规范** - 遵守 [Code of Conduct](https://github.com/Claudate/project-multilevel-index/blob/master/CODE_OF_CONDUCT.md)
