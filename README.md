## Usage

These skills can be used with Claude Code by:

1. **Direct copy**: Copy SKILL.md to your project's `.claude/skills/` folder
2. **Plugin install**: Use `/plugin marketplace add <path>` for marketplace-enabled skills
3. **Reference**: Use as templates for creating your own skills
